# Estudiantes App

Aplicación web desarrollada con FastAPI (backend) y JavaScript (frontend).

## Funcionalidades
- Autenticación con OTP
- CRUD de estudiantes
- Despliegue en la nube (Render)

## Tecnologías
- FastAPI
- JavaScript
- HTML/CSS
- Render
